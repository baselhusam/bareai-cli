Install

Pick a path for your machine. Full docs and flags live in the repo README.

install.sh 
curl -fsSL https://raw.githubusercontent.com/baselhusam/bareai-cli/main/scripts/install.sh | bash
source ~/.zshrc && bareai version
Homebrew 
brew tap baselhusam/tap
brew trust baselhusam/tap
brew install bareai
bareai version
winget 
winget install baselhusam.bareai
bareai version
Debian / Ubuntu 
curl -1sLf 'https://dl.cloudsmith.io/public/baselhusam/bareai/setup.deb.sh' | sudo bash
sudo apt update && sudo apt install bareai
bareai version

What it does

One question on a single machine: what is this box doing right now?

  • Collect, don’t mutate

    Host, GPU (NVIDIA / AMD / Apple), Docker, and LLM runtimes — inspect and probe only. Missing pieces degrade cleanly.

  • Correlate the stack

    Join endpoint → container → PID → GPU index → VRAM so you can see which model is burning which accelerator.

  • CLI, TUI, or JSON

    Tables for SSH, a live Bubble Tea dashboard for day-to-day, and --json for scripts and agents.
